Руководство пользователя для программы "Обработка заявок"
Введение
Программа "Обработка заявок" предназначена для управления заявками на техническое обслуживание. Она позволяет пользователям добавлять, редактировать, удалять и просматривать заявки, а также управлять списком пользователей, типов неисправностей и статусов заявок.

Установка и запуск
Установка программы:

Загрузите установочный файл с официального сайта.
Запустите установочный файл и следуйте инструкциям на экране для завершения установки.
Запуск программы:

Дважды щелкните по значку программы на рабочем столе или найдите программу в меню "Пуск".
Авторизация
При первом запуске программы вам будет предложено ввести логин и пароль для входа.

Ввод учетных данных:

Введите ваш логин в соответствующее поле.
Введите ваш пароль в поле для пароля.
Нажмите кнопку "Войти".
Сообщения об ошибках:

Если введены неверные учетные данные, вы увидите сообщение "Пользователь с указанными данными не найден!".
Если у вас нет доступа к программе (например, ваша роль ID равна 4), вы увидите сообщение "У вас нет доступа к этому приложению!".
Основные функции
Просмотр заявок
После успешного входа в систему откроется главное окно программы, где будет отображен список всех заявок.

Фильтрация и сортировка:
Вы можете использовать фильтры для поиска заявок по различным критериям, таким как статус заявки, дата добавления и т.д.
Для сортировки заявок нажмите на заголовок соответствующего столбца.
Добавление новой заявки
Нажмите кнопку "Добавить заявку".
В открывшемся окне заполните все необходимые поля:
Номер заявки: Уникальный идентификатор заявки.
Дата добавления: Дата, когда заявка была добавлена.
Модель: Модель оборудования, требующего обслуживания.
Оборудование: Тип неисправности, выберите из списка.
Описание проблемы: Подробное описание проблемы.
Клиент: Выберите клиента из списка пользователей.
Статус: Выберите текущий статус заявки.
Нажмите кнопку "Сохранить", чтобы сохранить заявку, или кнопку "Отменить", чтобы закрыть окно без сохранения изменений.
Редактирование заявки
Выберите заявку из списка.
Нажмите кнопку "Редактировать заявку".
В открывшемся окне внесите необходимые изменения.
Нажмите кнопку "Сохранить", чтобы сохранить изменения, или кнопку "Отменить", чтобы закрыть окно без сохранения изменений.
Удаление заявки
Выберите заявку из списка.
Нажмите кнопку "Удалить заявку".
Подтвердите удаление заявки в появившемся диалоговом окне.
Управление пользователями
Просмотр пользователей:

Перейдите в раздел "Пользователи", чтобы увидеть список всех пользователей.
Добавление пользователя:

Нажмите кнопку "Добавить пользователя".
Введите информацию о пользователе: фамилия, имя, отчество, телефон, логин, пароль и роль.
Нажмите кнопку "Сохранить", чтобы сохранить нового пользователя.
Редактирование пользователя:

Выберите пользователя из списка.
Нажмите кнопку "Редактировать".
Внесите необходимые изменения и нажмите кнопку "Сохранить".
Удаление пользователя:

Выберите пользователя из списка.
Нажмите кнопку "Удалить" и подтвердите удаление в появившемся диалоговом окне.
Управление справочниками
Типы неисправностей:

Перейдите в раздел "Типы неисправностей".
Добавляйте, редактируйте или удаляйте типы неисправностей аналогично управлению пользователями.
Статусы заявок:

Перейдите в раздел "Статусы заявок".
Добавляйте, редактируйте или удаляйте статусы заявок аналогично управлению пользователями.
Выход из программы
Для выхода из программы закройте главное окно. Все несохраненные данные будут потеряны.

Заключение
Данное руководство охватывает основные функции программы "Обработка заявок". Если у вас возникли дополнительные вопросы или проблемы, обратитесь в службу технической поддержки.
